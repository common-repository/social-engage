Get your social network shares, likes, tweets, and view counts of posts from different social networks.

== Description ==
Get your social network shares, likes, tweets, and view counts of posts from different social networks including Facebook, Twitter, Pinterest and LinkedIn. Track the performance of your pages and know your best performing posts.

Watch how to install and configure Social Engage here.

== Features ==

1. Social counts from Facebook, LinkedIn, Twitter and Pinterest
2. Dashboard with social counts
3. Monitoring of popular posts from social networks
4. A Widget to configure most shared posts in social networks
5. See your most shared posts in social media using a sidebar widget

== Installation ==
1. Upload "init.php" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate to the Social Engage menu in the WordPress admin.

== Frequently Asked Questions ==
= How to get social counts using Social Engage =
Once you install this plugin, the plugin by itself monitor the social networks for your posts by using a WP Cron Job. You can see the social counts in the dashboard page.

= What about Social Engage? =
Want to know your top performing posts in social media. This tool will help you with the social media counts.

= Will this plugin work with WordPress versions less than 2.7? =

No. WordPress 2.7 changed the way options were updated, so the options page will only work on 2.7 or later.
